# Automation Control Center

A simple automation dashboard built using **HTML, CSS, and Python** to manage and monitor automation tasks.

## 🚀 Features

* Clean and responsive UI
* Automation monitoring dashboard
* Integration-ready with Python scripts
* Simple and lightweight design

## 🛠️ Technologies Used

* HTML5
* CSS3
* Python

## 📂 Project Structure

```
project-folder/
│
├── index.html
├── style.css
├── script.js
└── app.py
```

## ▶️ How to Run

1. Clone the repository
2. Open the project folder
3. Run the Python script (if backend is used)

```
python app.py
```

4. Open `index.html` in your browser.
